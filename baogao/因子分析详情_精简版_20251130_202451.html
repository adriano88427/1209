评分标准说明

本报告采用“分维度评分 + 动态可靠性加权”方案： 1. **基础表现（日度 IC / IR / 显著性）** - IC：|IC|≥0.05 → 100 分，0.03 → 90，0.02 → 75，0.015 → 60，0.01 → 45，其余 20； - IR：|IR|≥1.5 → 100，1.0 → 90，0.7 → 80，0.5 → 70，0.3 → 55，0.1 → 40； - 统计显著性：p<0.01 → 100，0.01~0.05 → 85，0.05~0.1 → 70，≥0.1 → 40； - 多空收益 / 胜率：收益≥4% → 100，3% → 90，2% → 80；胜率≥65% → 100，50% → 65。最终基础得分 = 40%*IC + 20%*IR + 20%*p 值 + 20%*(收益/胜率均值)。 2. **整体样本表现（全样本 IC / IR / p 值 / 样本量）** - 指标来自 `extra_stats['overall_metrics']`；IC / IR 与基础层级一致； - 样本量：≥800 → 100，600 → 95，400 → 90，300 → 85，200 → 75，120 → 65，80 → 55，40 → 45； - 综合得分 = 45%*整体 IC + 20%*整体 IR + 15%*p 值 + 10%*多空收益 + 10%*样本量。 3. **滚动稳定度（Rolling IC）** - 统计每个滚动窗口的 CV / 半衰期 / MAIC，分别使用 `_score_cv/_score_half_life/_score_maic`； - CV≤0.3 → 100，0.6 → 60，>0.8 → 20；半衰期≥20 → 100，≤5 → 40；MAIC≥0.05 → 100，0.02 → 65； - 各窗口得分求平均得到滚动得分。 4. **时序一致性** - Lag1 自相关最佳区间 0.4~0.8； - 趋势相关系数 |r|≤0.2 保持高分，|r|>0.5 降至 20； - 换向次数 1~3 次最优，>6 次降至 45；结合均值回归强度、排名波动等指标加权。 5. **样本稳健性（抽样敏感度）** - 80%/90%/100% 抽样的 IC 均值、标准差、IQR、成功率分别评分：std≤0.003 → 100，0.01 → 65，>0.02 → 35；成功率≥95% → 100，<80% → 35； - 跨样本方差≤0.01 → 95，0.02 → 60，>0.05 → 30；平均后得到样本稳健性。 6. **动态权重与可靠性** - 名义权重：基础 35% / 整体 25% / 滚动 15% / 时序 12.5% / 样本 12.5%； - 每维度计算 0.3~1.0 的可靠性（依据样本量、缺样、集中度、滚动 CV 等）； - `final_weight = base_weight * clamp(reliability, 0.4~1.6)`，若可靠性 <0.35 则剔除；归一化后限制在 5%~55%。 7. **综合评分与评级** - 最终得分 = Σ(维度得分 * 动态权重)，保留 2 位小数； - 评级：A+ ≥90，A ≥85，A- ≥75，B+ ≥65，B ≥55，C+ ≥45，否则 C； - 正/负向因子根据评级给出建议权重与权重调整说明。 8. **低样本与风险提示** - 若触发低样本模式（avg<4 或缺样>20%），报告将提示“低样本量模式，窗口=3 日，min=2”，并上调稳健性权重； - 滚动/时序可靠性为“低”时，报告内会标注“稳健性权重不足，建议谨慎使用”。 9. **投资建议提示** - A± 因子建议 10%~25% 权重，B 级 3%~10%，C 级主要用于辅助或对冲； - 负向因子按反向 IC 评分，报告会给出反向胜率与权重提示。
