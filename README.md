# 因子分析项目

这是因子分析项目的完整代码仓库。

## 项目文件

本仓库包含所有项目文件，核心模块包括：

- `yinzifenxi/`: 核心因子分析代码模块
- `memory/`: 项目文档和分析报告
- `tools/`: 实用工具脚本
- `jietu/`: 项目截图文件

## 排除的目录

根据要求，以下目录被排除：
- `baogao/`: 报告输出目录
- `shuju/`: 数据输入目录

## 项目说明

这是一个完整的因子分析系统，包含单因子、双因子分析功能，支持参数化和非参数化分析。
